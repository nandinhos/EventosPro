# Contexto do Projeto: Escolha de Frontend

- **Status**: Aplicação baseada em Blade.
- **Nota**: O Filament está instalado no projeto, mas NÃO está sendo utilizado atualmente para as funcionalidades.
- **Ação**: Todas as novas funcionalidades e mudanças de UI devem ser implementadas usando templates Blade e controllers Laravel padrão, a menos que explicitamente instruído o contrário.
- **Futuro**: Esta regra deve ser atualizada caso o projeto migre para o Filament.
